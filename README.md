
# Proyecto Final LMS - Base de Datos

Este repositorio contiene el diseño e implementación de una base de datos para un sistema LMS orientado a la gestión académica, tutorías, roles, permisos, sesiones, inscripciones, asistencias y evaluaciones.

El proyecto fue desarrollado en **SQL Server** usando lenguaje **T-SQL**.

## Descripción del proyecto

El sistema LMS permite administrar diferentes áreas importantes de una plataforma académica, tales como:

- Gestión de usuarios.
- Gestión de roles y permisos.
- Registro de materias.
- Administración de tutores.
- Validación de tutores.
- Gestión de horarios.
- Creación de sesiones académicas.
- Inscripción de estudiantes.
- Registro de asistencias.
- Evaluación y retroalimentación de sesiones.
- Auditoría de cambios y accesos.
